NAME
====

App::Csv2Vcf - Conversion from tab-separated csv files to vcf format

SYNOPSIS
========

```bash
csv2vcf -i input-file.csv -o output_file.vcf
```

DESCRIPTION
===========

App::Csv2Vcf is a simple converter from CSV to VCF
